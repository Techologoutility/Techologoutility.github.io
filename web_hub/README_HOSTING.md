# Jatin Utility Hub (Free Hosting Setup)

## Local run
- Double-click `run.bat` inside `web_hub`
- Or open `web_hub/index.html` in any browser

## Share it with others (GitHub Pages - Free)
1. Create a GitHub account if you do not have one
2. Create a new repository (example: `jatin-utility-hub`)
3. Upload this project folder to that repository
4. Make sure your default branch is `main`
5. Push your code (the workflow in `.github/workflows/deploy-web-hub-pages.yml` will deploy automatically)
6. In GitHub repo settings, open `Pages` and ensure source is `GitHub Actions`

## Public link format
- `https://YOUR_GITHUB_USERNAME.github.io/REPOSITORY_NAME/`
- Example: `https://jatin123.github.io/jatin-utility-hub/`

## Notes
- The website files are in `web_hub/`
- You can keep adding new tools in `web_hub/index.html`, `web_hub/style.css`, and `web_hub/script.js`
